# HWPO — transcripciones de referencia

Fuente: HWPO Training (Mat Fraser). Programas transcritos: **Strong 2.0**,
**Foundations**, **Flagship 60 2.0**. Material crudo para destilar exemplars.

## ADN observado (resumen para destilación)

- **Estructura del día:** WARMUP → PRIMARY STRENGTH → SECONDARY/SKILL → ACCESSORY
  → METCON (frecuentemente OPCIONAL) → CARDIO/BONUS (opcional). Fases de 4 semanas,
  semana 4 = deload. "Working Max" = 90% del 1RM real.
- **Fuerza:** acumulación de volumen + tiempo bajo tensión; reps moderadas-altas
  con RPE; builds técnicos por sensación ("build to a heavy technical 10 // RPE 8")
  en vez de % rígido; descanso explícito por set.
- **Accesorios:** NO son relleno, "DRIVE progress". Tempos (@2020), :30 max reps,
  cerca del fallo, "buscá el pump", reps que caen set a set.
- **Metcon:** opcional y dosificado para no interferir con la fuerza; grind pace;
  carries + burpees; pesos livianos para ir unbroken.
- **Cardio:** opcional, Zona 2 en máquina + intervalos medidos; "no afecta la
  progresión de fuerza"; cerrar con un all-out.
- **Voz de las NOTES:** coaching por movimiento, explica el POR QUÉ, da opciones
  de escalado y subs, tono directo.

## Transcripciones crudas

### Strong 2.0 — intro (Base phase)

> Phase "Base": laying a strong foundation by accumulating overall volume and
> increasing time under tension each week. Moderate-high reps on main lifts,
> timed sets on accessories. Phase One = two four-week blocks. Build the
> foundation for big one-rep maxes. Accessory work DRIVES progress, not supports.
> Cardio optional (2 machine sessions, 1 metcon, 1 EMOM/week) written to not
> affect strength. Wk1-3 Build, Wk4 Deload, Wk5-6 Build//Heavy Single,
> Wk7 Heavy Triple, Wk8 Deload+transition.

### Strong 2.0 — Bench day

```
WARMUP: 3:00 machine/run + 2-3 sets [10/10 Pec Stretch, 10/10 Trap Stretch, 5 Prone IYT], rest 1:00
PRIMARY STRENGTH: Bench Press 1x5 @33 / 1x5 @39 / 1xMax @45 //RPE8 / 1xMax @33 //RPE9, rest 2-3'
  + Strict Press 5x10 @30, "no bounce off collarbones, reset each rep"
ACCESSORY: KB Seal Row 4x8-12 (close to failure) · Band Pull Apart 100 for quality
  · 3 sets @RPE9: 15-20 Rolling DB Tricep Ext + 15-20 DB Hammer Curl ("big pump")
METCON (opt): For Time cap 11': 40 Burpee, 200m Farmers Carry 2x35/23lb, 20 Burpee, 100m Farmers Carry
```

### Strong 2.0 — Deadlift day

```
WARMUP: 3:00 machine/run + 2-3 sets [10/10 Deadbug, 12 Goblet Squat, 20 Walking Lunge w/DB Deadlift]
PRIMARY STRENGTH: Deadlift — 10-12' build to heavy technical set of 10 //RPE8, + 3x5 @ same weight
  ("heavy technical" = challenges posture without crushing; ~91-105kg reference, not law)
ACCESSORY: Glute Ham Raise 4x5-10 · Bent Over Row 3x10 @RPE8 · Shrug 4x15 · 3:00 Plank Matrix
METCON (opt): AMRAP 12: 12/9 cal Fan Bike, 50' Sandbag Carry 150/100lb, 150' Farmers Carry 2x70/53lb
  ("steady grind, RPE7 on bike")
```

## Lote 2 (transcripciones 9-14) — patrones nuevos

- **SUBSTITUTION & SCALING GUIDE (nuevo, destilado a facet `scaling`):** subs por
  ratio explícito — Thruster→Double DB Thruster 1:1 / Air Squat 2:1 / Tuck Jump 1:1;
  Row→Bike Erg 1:1, Echo 1:2, Ski 1:1.5; Run→Row 1:1.5, Echo 1:1.25, Shuttle(10m=1rep);
  GHD Sit-Up→Weighted AbMat 1:1 / Dual Leg Raise; HSPU→Z-Press(DB) 1.5:1 / DB Push Press / Push-Up;
  Rope Climb→Push-Up; DU→Single-Under 1.5:1 / Plate Hop 1:1 / Lateral Line Hop 1:1;
  Muscle-Up→Heavy Pull-Up+C2B / Ring Pull-Up+Ring Dip; Bar MU→Burpee Pull-Up 1:1;
  Bike Erg→Row/Ski/Echo; Pistol→Shuttle Run; Burpee Box Jump Over→Step Over / Broad Jump 1:1;
  Clean→Power Clean (BB/DB).
- **Fuerza en olas** (nuevo ejemplo del mismo facet): Strict Press 1x5@RPE6 / 1x3@RPE7 / 1x1@RPE8.
- **Complejos de fuerza-gimnasia:** 6 sets [2 Bench + 4 Drop Push-Up]; 3 sets 5 Weighted Ring Dip @RPE7.5.
- **Metcon con carga descendente + carry entre sets** (refuerza el patrón): For Time 42-30-18 GHD
  Sit-Up (100ft Bear Hug Carry entre sets), rest 2:00, 30-20-10 Dual KB Deadlift 2x32/24; pesos por sexo.
- **AMRAP con pacing repetido:** 3 sets AMRAP 5: 200/150m Row, 15 Wall Ball, 200m Run — "consistencia,
  misma eficiencia; escalá a AMRAP 4 si no cerrás 3 sets".
- **Rest day explícito** (Flagship 2.0): días de descanso marcados en el calendario.
